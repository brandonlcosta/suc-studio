# SUC Route Builder

Local authoring tool for SUC Route Viewer.

## Phase 1: Scaffold

Minimal Electron + React app. No features yet.

## Setup

```bash
cd route-builder
npm install
```

## Development

```bash
npm run dev
```

This will:
1. Start Vite dev server
2. Compile TypeScript (main/preload)
3. Launch Electron with hot reload

## Build

```bash
npm run build
npm start
```
